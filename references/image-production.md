# 图文生图参考

## 视觉资产类型

公众号内容默认考虑 4 类图：

1. 封面图：吸引点击，表达冲突。
2. 正文信息图：解释核心逻辑，如对比、流程、指标、清单。
3. 图文卡片：适合社群/朋友圈/小红书二次分发。
4. 结尾系列图：沉淀栏目感，提示下一篇。

## 规格建议

- 公众号封面常用：`2.35:1` 横图，文字少，主体明确。
- 正文信息图：宽 1080px，长图高度按内容控制，移动端可读。
- 图文卡片：`3:4` 或 `1:1`，每页一个观点。
- 所有图片文字必须大、短、可读，不把整段文章塞进图里。

## 风格原则

- 健身房商业洞察：红黑白、数据感、门店场景、经营复盘风。
- 营养干货：干净明亮、食物真实、清单明确、少用医疗感。
- 热点讨论：事件符号 + 行业迁移，不做娱乐八卦感。
- 个人IP：保留统一色彩、栏目名、署名或固定结尾样式。
- 面向 30+ 高知用户的封面：优先使用有向往感、可代入的生活方式照片作为背景，如晨光厨房、安静运动角、精致外食选择、周末自我照顾场景；再叠少量标题文字。不要只做纯文字海报。
- 公众号封面必须做移动端完整性检查：人物、餐桌、食物、运动姿态、门店主体等关键画面不能被横向裁掉。若原图比例不适配，使用“完整图 fit inside + 模糊/柔和背景延展 + 标题区渐变遮罩”，不要为了文字区直接裁掉主体。

避免：

- 虚假前后对比图。
- 夸张医疗效果图。
- 过度 stock 感的健身人物。
- 图片中文字太密。
- 侵权品牌 logo 或未经授权的人像。

## 生图提示词模板

### 封面图

```text
公众号封面背景图，主题：[主题]，目标读者：[30+高知女性/健身房老板/减脂人群]，
画面元素：[用户容易带入且向往的生活场景]，
风格：[高级生活方式摄影/清爽营养科普/商业杂志纪实]，
留出干净区域给后期标题叠字，
无文字，无logo，无水印，不仿真人明星，不使用未经授权肖像。
```

### 正文信息图

```text
中文信息图，宽屏移动端可读，主题：[主题]，
结构：[对比/流程/三步清单/指标树/时间轴]，
包含文字：[列出每个模块短句]，
风格统一，留白充足，大字号，适合插入微信公众号正文。
```

### 图文卡片

```text
生成一组中文图文卡片方案，共 [5-9] 页，主题：[主题]，
目标读者：[读者]，每页一个观点，文字短，适合社群转发。
输出每页：页码、标题、正文短句、画面元素、生图提示词。
```

## 产出流程

1. 先确定图片在文章中的功能：吸引、解释、总结、转发。
2. 再写图片文案，不超过图片可读容量。
3. 再生成或制作图片。
4. 最后检查移动端可读性：文字不拥挤、对比足够、主体不糊。
5. 在手机宽度预览首屏，确认封面主体完整、标题可读、没有把向往感场景切碎。

如果用户要求“优秀生图能力”，优先交付完整视觉方案和提示词；有可用生图工具时直接生成图片并插入预览。
